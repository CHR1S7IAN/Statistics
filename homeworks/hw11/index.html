Detailed Explanation of Concepts

1. The Wiener Process (Brownian Motion)

The Wiener Process ($W_t$) is the continuous-time limit of a random walk. It is the fundamental building block of stochastic calculus. Its key properties are:

  • Start at Zero: $W_0 = 0$.
  • Independent Increments: Future movements do not depend on past movements (Markov property).
  • Gaussian Increments: The change $W_{t+dt} - W_t$ follows a Normal distribution $\mathcal{N}(0, dt)$.

2. The Stochastic Differential Equation (SDE)

We are simulating an Arithmetic Brownian Motion (ABM) described by the SDE:

$dX_t = \mu dt + \sigma dW_t$

  • Drift ($\mu dt$): The deterministic trend. If $\mu > 0$, the process tends to go up over time.
  • Diffusion ($\sigma dW_t$): The stochastic (random) part. $\sigma$ controls the volatility (noise).

3. The Numerical Method: Euler-Maruyama

Since computers cannot handle continuous time, we discretize the interval $T$ into $N$ steps of size $dt = T/N$. The Euler-Maruyama method approximates the next value as:

Xn+1 = Xn + μ·dt + σ·√dt·Z

Where Z is a random number generated from a Standard Normal Distribution ($\mathcal{N}(0,1)$).
Crucial Note: We multiply the noise by $\sqrt{dt}$, not $dt$, because the variance of Brownian motion scales linearly with time ($Var(W_t) = t$).

4. Generalization (The "Optional" Part)

The code is structured to handle general SDEs where drift and diffusion are functions of the current state, not just constants:
$dX_t = a(X_t, t)dt + b(X_t, t)dW_t$

While this simulation uses constant $\mu$ and $\sigma$, the underlying solver class supports complex models like Geometric Brownian Motion (where diffusion depends on $X_t$).
