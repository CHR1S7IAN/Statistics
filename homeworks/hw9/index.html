1. The Main Interpretations of Probability

There are several "schools of thought" on what probability fundamentally represents. Each provides a different lens for understanding uncertainty.

Classical (Laplace)

This is the intuitive, introductory definition. If an experiment has N equally likely outcomes, and an event A consists of k of these "favorable" outcomes, then the probability of A is:

P(A) = k / N = (Favorable Cases) / (Total Possible Cases)

Limit: This approach fails when outcomes are not equally likely or when the sample space is infinite.

Frequentist (von Mises)

The probability of an event is the limit of its relative frequency observed after repeating an experiment a very large (theoretically infinite) number of times under identical conditions.

P(A) = lim n→∞ (nA / n)

...where n is the total number of trials and nA is the number of times A occurred.

Limit: This is not applicable to unique, one-time events (e.g., "the probability of candidate X winning tomorrow's election"), as they cannot be repeated.

Subjective (Bayesian, de Finetti)

Probability is a measure of an individual's degree of belief that an event will occur, based on all available information. It is subjective, but must be *coherent* (to avoid guaranteed losses in bets, known as "Dutch books"). Probabilities are updated as new evidence becomes available, using Bayes' Theorem.

Geometric

An extension of the classical approach to continuous (and infinite) sample spaces. Probability is defined as the ratio of "measures" (length, area, volume):

P(A) = Measure(A) / Measure(Ω)

Limit: Can lead to paradoxes (like Bertrand's paradox) if the sample space and "random" mechanism are not precisely defined.

2. The Axiomatic Approach: A Common Foundation

The various interpretations are philosophically incompatible. The axiomatic approach, introduced by Andrey Kolmogorov in 1933, brilliantly resolves this conflict.

How? It ignores the question "What is probability?" (the semantics) and focuses instead on "How does probability behave?" (the syntax).

Kolmogorov established three simple rules (axioms) that any probability function P must obey. It doesn't matter if your probability comes from counting (classical), frequency (frequentist), or belief (Bayesian); as long as it satisfies these axioms, you can use the entire mathematical machinery of probability theory.

Given a sample space Ω (the set of all outcomes) and a family of events F (the subsets of Ω we assign probabilities to), a probability measure P is a function that satisfies:

  • Axiom 1 (Non-negativity): The probability of any event E is non-negative.
    P(E) ≥ 0 for every E ∈ F
  • Axiom 2 (Normalization): The probability of the entire sample space (the "certain event") is 1.
    P(Ω) = 1
  • Axiom 3 (Countable Additivity): For any sequence of mutually disjoint events (meaning Ei ∩ Ej = ∅ for i ≠ j), the probability of their union is the sum of their individual probabilities.
    P(E1 ∪ E2 ∪ ...) = P(E1) + P(E2) + ...

3. Probability and Measure Theory

Kolmogorov's work established that probability theory is, in fact, a specialized branch of measure theory. Measure theory is the mathematical field that generalizes the concepts of length, area, and volume.

The connection is a direct one-to-one mapping. A probability space (Ω, F, P) is simply a measure space (X, M, μ) where the total measure of the space is 1.

Measure Theory Concept Probability Theory Concept Explanation
Set (X) Sample Space (Ω) The set of all possible outcomes (e.g., {Head, Tail}).
σ-Algebra (M) σ-Algebra (F) The family of events. The collection of subsets of Ω to which we can assign a probability.
Measure (μ) Probability Measure (P) A function P: F → [0, 1] that assigns a probability to each event.

Random Variables as Measurable Functions

In this framework, a Random Variable X (which assigns a number to an outcome, e.g., X(Head) = 1) is formally defined as a measurable function.

A function X: Ω → R is measurable if, for any real number x, the set of outcomes ω such that X(ω) ≤ x is an event in our σ-algebra F.

{ω ∈ Ω | X(ω) ≤ x} ∈ F

This formalism ensures that it is "legal" to ask questions like "What is the probability that X is less than or equal to x?".

4. Derivations from the Axioms

Using only the three axioms, we can derive all other rules of probability.

Derivation: Inclusion-Exclusion Principle (for n=2)

Goal: Prove P(A ∪ B) = P(A) + P(B) - P(A ∩ B).

  1. We can write the union A ∪ B as the union of two disjoint sets: A and (B \ A) (elements in B but not in A).
  2. By Axiom 3 (Additivity), since A and (B \ A) are disjoint:
    P(A ∪ B) = P(A) + P(B \ A) --- (Eq. 1)
  3. Now, consider event B. We can also split it into two disjoint parts: (B ∩ A) and (B \ A).
  4. Applying Axiom 3 to B:
    P(B) = P(B ∩ A) + P(B \ A)
  5. Rearranging this, we get an expression for P(B \ A):
    P(B \ A) = P(B) - P(B ∩ A) --- (Eq. 2)
  6. Finally, substitute (Eq. 2) into (Eq. 1):
    P(A ∪ B) = P(A) + [ P(B) - P(B ∩ A) ]
  7. P(A ∪ B) = P(A) + P(B) - P(A ∩ B) (Q.E.D.)

Derivation: Subadditivity (Boole's Inequality)
Case 1: Finite (for n=2)

Goal: Prove P(A ∪ B) ≤ P(A) + P(B).

  1. From the Inclusion-Exclusion principle:
    P(A ∪ B) = P(A) + P(B) - P(A ∩ B).
  2. From Axiom 1 (Non-negativity), we know that P(A ∩ B) ≥ 0.
  3. Therefore, -P(A ∩ B) ≤ 0.
  4. If we remove this non-positive term from the right side, the right side can only increase or stay the same:
    P(A) + P(B) - P(A ∩ B) ≤ P(A) + P(B).
  5. Combining steps 1 and 4 gives: P(A ∪ B) ≤ P(A) + P(B) (Q.E.D.)
Case 2: Countable (General)

Goal: Prove P(Ui=1∞ Ai) ≤ ∑i=1∞ P(Ai).

  1. We construct a new sequence of sets Bi that are disjoint and have the same union as the Ai.
    • B1 = A1
    • B2 = A2 \ A1
    • B3 = A3 \ (A1 ∪ A2)
    • ...
  2. By construction, the Bi sets are mutually disjoint, and Ui Bi = Ui Ai.
  3. Applying Axiom 3 (Countable Additivity) to the disjoint sequence Bi:
    P(Ui Ai) = P(Ui Bi) = ∑i P(Bi).
  4. By construction, every Bi is a subset of its corresponding Ai (Bi ⊆ Ai).
  5. By the monotonicity property (derived from the axioms, if C ⊆ D then P(C) ≤ P(D)), we have:
    P(Bi) ≤ P(Ai) for all i.
  6. Substituting this inequality into the sum from step 3:
    ∑i P(Bi) ≤ ∑i P(Ai).
  7. Combining steps 3 and 6, we get the result:
    P(Ui Ai) ≤ ∑i P(Ai) (Q.E.D.)
