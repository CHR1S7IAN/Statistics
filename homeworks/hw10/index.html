Detailed Explanation of Concepts

1. The Stochastic Process (Poisson Process)

The mathematical model described is the Poisson Process. It is the standard model used to describe events that occur spontaneously and randomly in continuous time, such as incoming calls at a call center, the radioactive decay of an atom, or the arrival of customers in a queue.

2. How the Simulation Works (The Approximation)

The suggested method uses the Bernoulli Approximation to handle continuous time on a computer (which operates in discrete steps):

  • Time Division: The total interval T is divided into a grid of n tiny sub-intervals.
  • The "Coin Toss": In each interval, a Bernoulli trial is performed. The probability of an event occurring is very low: p = λ/n.
  • Convergence: Mathematically, summing the successes of these n intervals gives a Binomial Distribution. As n tends to infinity (becomes very large), the Binomial converges to the Poisson Distribution.

3. Fundamental Theoretical Properties

Three key properties emerge from the text:

  • Independence: What happens in one sub-interval does not influence the others. An event that just occurred does not change the probability of the next one.
  • Uniformity (Uniformly in time): The probability of an event occurring is constant throughout the entire interval T. There are no "peak times".
  • Memoryless: The process "does not remember" how long it has been waiting. The time elapsed since the last event provides no information on how long remains until the next one.

4. The Meaning of λ (Lambda)

The parameter λ represents the intensity rate (rate parameter).

It indicates the average number of expected events per unit of time T. For example, if we are simulating 1 hour (T=1) and set λ = 10, we expect an average of 10 events to happen in that hour. In the simulation, we divide this rate by n to get the probability for each single micro-interval.
